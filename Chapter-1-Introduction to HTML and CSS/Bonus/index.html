HTML & CSS Quick Reference Guide

This guide provides a quick reference to essential HTML tags and CSS properties with examples.

HTML Tags

  • <h1> - <h6>: Headings for different levels.
  • <p>: Paragraph text.
  • <a>: Hyperlinks.
  • <img>: Images.
  • <ul> / <ol> / <li>: Lists.
  • <div>: Generic container.
  • <span>: Inline container.
  • <code>: Inline code display.
  • <pre>: Preformatted block of code.
  • <br>: Line break.

<h1>My Page</h1>
<p>Welcome to my site!</p>
<a href="https://example.com">Click here</a>

CSS Properties

  • color: Sets text color.
  • background-color: Sets background color.
  • font-size: Sets font size.
  • margin: Sets outer spacing.
  • padding: Sets inner spacing.
  • border: Sets border style.
  • text-align: Aligns text (e.g. left, center).
  • font-family: Sets font type.

body {
    background-color: #f0f0f0;
    color: #333;
    font-family: Arial, sans-serif;
    padding: 20px;
}
